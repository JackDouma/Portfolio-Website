Software Engineering

Taken: Fall 2023

An introduction to a spectrum of software engineering concepts: software requirement and specification, project planning and design, software development and testing, software process management, and software maintenance. Several of these topics of software design and testing will be explored in detail. Object-oriented methodology and UML will be used. A team project and class discussion will be an integral part of the course.
